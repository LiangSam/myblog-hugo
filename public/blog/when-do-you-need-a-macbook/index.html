什么情况下你才需要一台MacBook？

最近有朋友问我这件事，他是想买一台MacBook，可是很纠结，不知道要不要买。他想买的原因是过年回家可以带上去玩玩《小丑牌》（Balatro）。

我看了一下，Steam页面上确实显示可以安装到MacBook上玩《小丑牌》。但如果原因非常单一，需求也很明确，在这种情况下，其实不太适合买MacBook。

在我看来，MacBook 比较适合创作型的工作，如码字、编程、搞艺术、剪辑视频等等（但也不完全精准，比如律师就很合适用 MacBook）。

这类创作不挑时段、不择场合，只要灵感来了都可以尝试。无论是在高铁上、飞机上，还是出门旅游，一台轻薄便携、续航强的 MacBook 就是最佳选择，而不是一台厚重、笨重的 Windows 台式机。

当然，Windows 笔记本也可以用，但就我个人来说，已经吃够了 Windows 的苦。因为之前的电脑用时间长了之后，就必须插着电源才能使用，电源一旦拔掉，很快就会关机。

如果你问我 MacBook 适合什么样的人，我觉得只要你满足两个条件：一是不打游戏，二是之前没用过，那你都可以去试一试。它给你的体验跟 Windows 完全是天差地别。如果说 Windows 属于「能用」，那 MacBook 就属于「精致地能用」。用罗永浩的话来说，这叫「漂亮得不像实力派」。

但如果你只是想着临时用一下，或者以为有了它就能写更多字、创作更多东西，那还是不能完全指望它。以我为例，在使用 MacBook 之前，我已经用 Windows 电脑写了一年多的文章，所以买到它之后，无非是把这个习惯迁移过去。中间虽然有对工具的适应期，但核心需求是不变的。

只是 MacBook 帮我把使用场景从家里拓展到了图书馆和高铁。不出意外的话，今年过年还能拓展到老家。我甚至能想象到，回到老家之后，我可以用 MacBook 来做一些什么。

首先最重要的应该是码字。众所周知，我是一个每天不写作就会浑身难受的人，需要用码字才能证明我的存在感。如果要是断更一天，我的朋友肯定都会想：这小子是不是出意外了？

去年的时候，虽然我也带了笔记本电脑回家，但由于太过笨重、手感糟糕，码字起来非常不便。过年那几天，我大部分时间都在外面，所以很少用它来码字，大多时候是在手机上打字后再修改。文章质量不能说差，只能说水到没边了。

现在有了 MacBook 之后，它只需要一根 Type-C 的手机充电线就够了，不需要额外再带一个非常笨重的充电器（比如我之前提到的那台 Windows 笔记本电脑）。

那时候我就在想，如果我有一台 MacBook 就好了。如今我真的实现了.

其次是看影视剧。跟之前的 Windows 笔记本电脑相比，用 MacBook 会更加享受。

因为那台笔记本只是 1080P 的屏幕，而且质量很差；MacBook 的像素肯定要好不少，画面干净通透，能享受到非常棒的观影体验。再加上它的硬盘是 512GB 的，下载几部 4K 影视剧完全不在话下，不用担心把我的硬盘给撑爆了。

最后就是打游戏。我可以下载 Steam，把我吃灰的好几个游戏（尤其是支持 macOS 的游戏）下载下来，痛痛快快地玩一段时间。

如此一来，在一个位置偏僻、寒冷，也没有太多娱乐方式的贵州农村，我靠着一台笔记本电脑，就能尽情享受一个独属于自己的小天地。

写到这里，好像都是在聊春节，那么跟 MacBook 有什么关系呢？

在我看来，它可以帮你切换场景。

一个是设备的切换。比如在使用 Windows 电脑时，我会忍不住去打开游戏，做一些无所谓的事。但在用 MacBook 的时候，游戏这方面已经完全断绝了。我没有在 MacBook 上装 Steam，至少对我来说，它的用途很简单：要么码字、观影，或者是看网页。

另一个是环境的切换。这意味着我可以从家里走出来，不再局限于 Windows 台式电脑。这也是为什么很多出差党会买一台 MacBook 的原因，因为它可以把场景切换出来，方便他们在高铁上出差的时候使用，而且便携轻薄、续航很长。

从这个角度来说，其实你用 iPad 也行，因为它更便携，适用的场景会更多。

可因为它的系统是 iPadOS 而不是 macOS，在软件和功能上无疑是受限的。如果你想要码字的话，还得额外配一块键盘，这对我来说非常繁琐，还不如用一台 MacBook 来的干脆。

因此，买不买 MacBook 其实不能看别人的建议，而应该看你的需求。

你需要考虑以下两点：你需要它来做什么？是否需要拓展更多的场景？

想清楚了这一点，你也就知道到底该不该买了。

我甚至觉得我应该早一点买。但也许只有买得够晚，我才知道它有多适合我。以上就是我对于该不该买 MacBook 的一个回答。
