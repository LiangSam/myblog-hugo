用了三个月 MacBook 后，我的 Windows 电脑成了“吉祥物”

2025-12-03

我使用 MacBook 已经三个多月，这期间，Windows 使用次数越发地少。关于使用体验，我在这篇文章《用了8年Windows的我，为什么要买MacBook？》中有过详尽描述。

那时我仍处于尝鲜期，觉得 macOS 上的一切都觉新鲜。它精致、漂亮、便携和长续航，后因生物污染，不得不快速脱手，及时止损，过段时间趁着国补，买一台全新 MacBook Air。

现在，我大抵度过蜜月期。我手里的 MacBook，系统全英文（方便按键找东西），安装 LaunchBar，付费 Keyboard Maestro（还想买 DEVONthink，但 1000 多元的价格，感觉我不用也行），快捷功能使用娴熟，能够满足我日常需求。

时间是零和博弈，你用来刷短视频，就没法同时看网文；电子设备的使用也是如此。由于我天天用 MacBook，最后竟有一周没用 Windows，甚至找不到一个特别的理由来打开它。

什么时候会用到 Windows 呢？只有玩游戏，剪视频，打印文章，下载影视剧，那我才会想起它。

可问题是，我很久没有打游戏，也没用这电脑剪公司视频，打印文章次数渐少，硬盘囤的几十部影视剧都没看，只顾着刷短剧和《喜人奇妙夜 2》。

于是，这台一开始被当宝贝一样用的 Windows 电脑，此刻就像吉祥物。它证明我年轻过，有过打游戏的时光，陪我搬了两次家，也遇到过无法开机的毛病，最后把内存条压紧实才顺利点亮。

Windows 就是这样，你用着用着，就不知道会遇到什么问题，只有通过检索信息，才找到解决方法。比如广受诟病的 C 盘爆满，我的解决方案是，直接用一根 1TB 的固态硬盘做 C 盘，不分区，游戏软件全往里面装，要见底了就删一个单机游戏，很快就有盈余空间。

MacBook 不会，一来我没想过要在上面打游戏，二来硬盘只有 512GB，我下载东西也会克制，用完即删。用到现在，也就遇到一次系统卡死，我常用的软件非常少，Safari、Obsidian、微信、NetNewsWire、网易云音乐等，其他就是辅助类软件，所以系统算干净。

这样一看，我似乎不用 Windows 也行，可以趁现在固态硬盘涨价卖掉。

最后想了想，还是作罢。我很难从理性角度来剖析这件事，总觉得我有一天用得上，需要剪视频打游戏，保留它是在保留生活的可能性，和过去的痕迹，而且还能和 MacBook 换换口味。

它们毕竟只是数码产品，还是要看你希望用它们来做什么，创造一些什么有意思的东西，这种时候，有什么是必须用 MacBook 才能解决的，又有什么，是非 Windows 不可的。

因为对于别人来说，他们其实并不在乎你用什么设备，不关心文字是用什么轴的机械键盘打出来，音乐从哪个音箱中冒出，视频又在多少分辨率的显示器呈现。

用什么设备根本不是问题，你为什么使用它，才是值得思考的。

那么你猜这篇文章，是在 Windows 上写的，还是 MacBook？
