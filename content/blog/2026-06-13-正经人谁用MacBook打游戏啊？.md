上周我的游戏瘾犯了，打开 Windows 电脑，扫了一圈游戏，发现《巴别塔圣歌》有点意思，我看评论都说不错，遂买之。

我非常欣喜地打开游戏，玩了半小时。你别说，还挺新鲜，就是根据别人的话来揣测每个字词是什么意思，那些话都跟乱码似的，但能通过情境揣测出来。后来我卡在一个关卡，于是退出游戏，电脑关机，熟练地把显示器切换回 MacBook。

其实我想找找在 macOS 上能玩的游戏，但这个行为又显得太诡异。不单单是我觉得，连游戏厂商也是这么想的。我查了一下 Steam 上的游戏，能在 macOS 上玩的大概只有 17%，但是能够适配 macOS 加苹果 M 系列芯片的又少一些，所以能玩的比较少。

我的 Steam 库里有 41 个游戏，其中支持 macOS 的有 15 个，但是绝大多数我已经体验过，比如 Hades、LIMBO、机械迷城、TOEM、这是我的战争等等，让我感觉还想继续玩的，就是无人深空、极乐迪斯科、Return of The Obra Dinn，但后两者对我的智商是一种考验，第一个又太花时间。

![Screenshot 2026-06-13 at 21.43.15.png](https://img.liangmouyin.com/2026/06/31746a6c544db19c09196e4be1f927bd.png)

简言之，我库里大部分都是好游戏。可正因为它们太重，我反而觉得打开之前需要做好心理建设，预留充足时间，才配得上沉浸地玩一会儿。这里面唯一通关的就是机械迷城，是在之前笨重的 Windows 笔记本上玩的，那时候我什么游戏都想玩，但买得起的不多。

此外我也纳闷，腾讯为什么不把《英雄联盟》上架 macOS。拳头运营的日服和美服都有 macOS 版本，但估计是因为国服内置了反作弊系统，移植到 macOS 成本高，再加上用户比例小，干脆就不做 macOS 版本，即 ROI 太低，不值得。

但这不意味着 macOS 用户没有娱乐需求，只是能用它来打大型游戏的人少之又少，还很容易让人觉得脑子有病。

我之前看过不少给大一学生推荐笔记本电脑的视频，说到 Windows 时弹幕还算和谐，顶多争论品牌，而一到 MacBook，很多人就说大学千万别碰 MacBook，因为很多专业软件不适配 macOS，到时候还要重新买电脑，而且还不能打游戏，买这干嘛。

大学用 Windows 难免需要折腾，但很多路别人都帮你走好了，比如大学我们需要安装 Adobe 软件，而老师连安装包都给我们准备好了。如果用 MacBook，只能自己上网找资料，也没法和别人开黑打游戏——不过当时隔壁寝室有个用 MacBook 的同学，偶尔回来我们寝室蹭游戏打。

苹果可能也意识到这个问题，前几年在 WWDC 高调演示苹果游戏，比如《死亡搁浅》《生化危机：村庄》，《赛博朋克 2077：终极版》还拿到今年的 Apple 设计大奖。诸如此类，可见苹果是想把 macOS 上的娱乐性补齐的。

但是适配无疑又会额外花时间，现在苹果 Keynote 的重心又转向了 AI，今年的 WWDC 浓墨重彩地说了苹果在 AI 上的突破，哪怕和国行没什么关系。

鉴于此，还是不能过于指望在 macOS 上能玩到更多游戏。我之前还想过要买 PS5 和 Switch，但也只是想想，我的 Steam 上还是有游戏能玩的，买了也是吃灰，倒不如找康哥借来耍耍，吃顿饭就能解决的事，没必要额外花钱。

写到这里的时候，我鬼使神差打开了 Steam，下载了《极乐迪斯科》。我一直觉得这游戏有点神神道道，里面的人物话特别多，角色跑来跑去也不知道在干嘛，好在它不吃操作，油画的设计风格非常独特，也适合在 MacBook 上耍。

希望我在未来几年，可以把这游戏打完吧，到时候就能写一篇《谁在 MacBook 上把极乐迪斯科打通关了啊？》，这太营销号了。
