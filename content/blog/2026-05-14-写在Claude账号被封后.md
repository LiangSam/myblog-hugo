下午我忽然发现笔记本连不上 Claude Code，不管我发送什么，它的回应都是「API Error: 400」。我从切换 Wi-Fi 到节点，结果还是一样，这时我打开浏览器，发现也进不去网站，最后心里音乐有一个猜测：我账号不会被封了吧？

果然，登入邮箱后，我翻了翻垃圾邮件，看到里面多出一封邮件，是下午三点半发过来的，发件人是 Anthropic 公司，说由于我的账户有可疑行为，违反了他们的使用政策，所以取消我对 Claude 的访问权限。

![](https://img.liangmouyin.com/2026/05/40aea47b1b7b47ab579b6c825b8bd51a.png)

F\*\*k Anthropic！

达摩克利斯之剑终于落下，我再也不用感到惴惴不安，所以必须要直抒胸臆表达我的感受，没有比这更管用的了。

我其实还买了 Pro 版会员，每月 20 美金，已经扣了三个月。到这里，我立刻上 Play Store 取消对 Claude 的订阅，没有必要再纠结。

其实我有想过为什么会被封：是我切换了节点？其实我一直用的是美国节点；是我说中文，Claude 觉得我是中国人必须封了？那我还能说什么语言；又或者是今天下午把 Claude Code 挂在后台，合上了笔记本的盖子，然后我的电脑运行一串神秘的代码，最后账号就被封了。

被封就被封了。

我前阵还用朋友推荐的接码平台额外注册一个 Claude 账号，现在倒是还能用，不过会员就不必再充了，免费版也够用。如果要写代码、建网站、修改博客，我还有 Codex 和 Gemini Cli，刚好这俩我都有会员，不至于乱了寸步难行，大不了我再用国内大模型的 API，活依旧能干——哪怕没 Claude 干得好，这我认。

关于数据，我也不是太操心，上周我导出过一次，也有个备份，只是少了这一周的而已。刚好我这一周也几近冷落了 Claude，而是用着 ChatGPT，干活也很麻利，帮我改了不少博客细节。

Claude 吸引我的，其实是它冷峻的风格。我三月前用的第一周，就非常欣赏，然后果断付费订阅；不过那感受也仅限当时，后来我很少再向之前那般探讨问题，而是让它不断做事，改博客、梳理 skills、给建议，等等。如果还想深入探讨，那再用我的新号即可。

当我看到被封提示的那一刻，内心犹有几分沮丧，但一切不可挽回，我很快接受这一现实。尽管我依然困惑，为什么 Anthropic 对我们的账号恶意这么大，一直往死里封，即便特朗普都来中国。

好在这些情绪，在我下班出门，看到一抹晚霞后，很快就烟消云散。晚霞绚烂，但我走到地铁口附近后，霞光也尽数敛去。而晚上我照例去喜家德吃一顿喜三鲜饺子，无比满足。

Anthropic 只能收走我对 Claude 的使用权，无法收走生活。使用这些工具，本来就是更为了更好的生活，但可没有它，我也还能吃饺子——我订阅 Claude 的六十美金，其实够我吃十几顿饺子了，而停止续费，我真的能多吃几顿饺子。

但我今天的问题是，下午吃得有点多，除了饺子，还额外点了一份红豆沙丸子，肚子撑得不行。我想到自从上次说我吃饺子上瘾后，有朋友晒图也吃上了喜家德饺子，而一位远在美国的读者兼作者也说准备试一次。

这才是让我觉得有趣的事情。

最后本想骂骂 Anthropic，但罢了。除了它，这世界多的是能写代码的对话框。既然 60 美金已经变成了胃里的饺子，那就这样吧。

在这个工具过载的时代，最好的往往没那么重要，能用的、不给人添堵的，才最重要。
