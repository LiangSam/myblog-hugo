我今天突然翻出老 Windows 笔记本，拿到手的感受是，我过去怎么能忍受这么蠢笨不堪的物件。

这台笔记本是游戏本来着，大学专业需要用 PR 和 AE 等专业软件，轻薄本难堪重任，所以选择性能较好的游戏本。这也是我的第一台电脑，陪我度过大学四年到工作后的三年，此前一直都觉得很耐用，唯一的毛病是电池不行，先是鼓包严重，我从网上买了一块新电池换上后，用了两三年，就必须要一直插电，拔掉电源不到 1 分钟就会关机。

但刚刚拿起笔记本时，我竟感觉陌生。

第一感觉是笨重，沉甸甸的，上秤后才知道有 4.5 斤重，而且还很厚。这不能给我任何踏实感，用了近半年的 MacBook Air 之后，我认为电脑就应该是这样轻盈便携，放在腿上码字不会感觉有负担，不想用时盒上盖可以轻松带走。

老笔记本的电源也很麻烦，它的电源适配器非常大块，像一块黑砖，插上后接口松动，开机就花了近半分钟。再看屏幕，黑边疆域辽阔，左右将近 2 厘米，上下 3.5 里面，中间是一块 16 英寸的屏幕。开机后，发现屏幕上有一个坏点，整体观看模糊，颜色寡淡。

一看开始菜单，里面有很多软件是两年前安装的。当时公司电脑实在难用，我只能学着康哥把闲置笔记本拿去用，这下工作效率提升不少。后来发现电脑总会莫名按几个键时，我觉得它差不多寿终正寝，就拿回出租屋。电脑里还有 Trae 等 AI 工具，但我现在基本都用不上了，只是卸载也很麻烦，需要挨个点击，最后太花时间，干脆放弃。

这时我才觉得用 MacBook Air 是一件多么幸福的事：续航可靠，单手开盖，纤薄轻巧（13 英寸重量为 1.24 千克），不用随时带着重重的充电器。再加上笔记本的触控板体验实在糟糕，按键松垮，功能单一，我居然习惯性用三指拖拉窗口，结果啥也没有，我才反应过来 Windows 没这功能，只能继续在这板子上搓动——如果说 MacBook 的触控板没有鼠标也能好用，那么我这台是没有鼠标还是别用了。

但是老电脑也不是一无是处，我在上面折腾过很多东西。比如重装系统，当时听说 Linux 很好用，装在另一个盘，导致卸载后电脑崩溃，我只能往返网吧，把系统写在 U 盘，再回家尝试；又比如 C 盘爆满，冬天后电脑莫名开不了机，正版软件太贵那么如何安装盗版，命令行可以做到哪些事，如何给电脑拆机加内存条、清灰、换电池，等等。

这对于大学时时间充裕，且喜欢折腾的我来说，它就像一个玩具。那时室友们的电脑也都是游戏本，所以丝毫不觉笨重，有时还能装进背包，在冬天踏着雪去上专业课，北方的夜来得更快，傍晚在大食堂吃晚卤肉饭，回到宿舍和室友开黑打游戏，熄灯后合上盖子，带着满足感躺回床上。

而现在已经过了折腾的年纪，我早已从 Windows 转向 macOS。诚然，这几年 Windows 笔记本不断迭代，好些品牌也能兼具轻薄和性能，续航也更加可靠，分辨率高，还是 Type-C 接口充电器，按理来说我选择它们会更稳妥。

只是恰好我还有 Windows 台式机，再买一台就是冗余，而且我还保留一些不信任：担心电池鼓包，电脑蓝屏，或者遇到奇奇怪怪的 BUG，那还是罢了。

现在，我也该找个机会，和这台笔记本告别了吧。
