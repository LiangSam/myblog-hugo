傍晚我去菜鸟驿站取上周买的最后两张 CD，到家后拆开塑封，取出罗大佑的《之乎者也》光碟，再放入 CD 机，播放，第一首《鹿港小镇》悠悠响起。

我开始听 CD 了。

不过我并非怀旧，毕竟我小时候家里也不听唱片，而是地摊上六元钱一张的盗版 DVD，有电视剧有电影，也有歌曲。然后插进播放器，就能放很久，但我父亲喜欢听歌曲，通常是刀郎、谢军、腾格尔，我敬而远之。

而我在 2026 年，数字音乐早已大行其道，我为什么非得折腾 CD 不可？

原因是上周五，我突然看到[和菜头教人如何选 CD 机](https://www.hecaitou.com/2026/05/cd-player-selection-reflections.html)，即避开所谓的网红 CD 机，选择靠谱的厂商，最后一定错不了。在这之前和菜头写过不少 CD 机的文章，但我委实看不懂，而那篇文章恰好让我想入坑。

我经常下班后打开音乐软件，随机播放一首歌，然后当作背景音乐，打扫房间，写作，逛网页。恰好我还折腾过本地音乐，那不妨再往前迈一步，整一台 CD 机试试？

随后我让 ChatGPT 帮我在山灵 EC Play、弗西 Merak、水月雨梦想碟机二代、飞傲 DM13 中为我推荐，最后建议我选择山灵 EC Play，因为它比较均衡。可在我看来，山灵这款是设计最为平庸的一个，其他三家在设计上各有巧思，均有天窗可以看到里面的 CD，非常精致。

但我相信 ChatGPT 的判断——然后在凌晨犹豫了一个小时左右，还是下单山灵 EC Play。

谁曾想，世界就是如此巧合，它好像听到我对 CD 感兴趣，小红书就像我推送深圳音乐展的信息，时间恰好在周末两天。知悉后我在周六通勤一小时过去，场内可谓人山人海，我听到旁边的人聊什么谁低频不行，哪款音质差，都是在我水平之上的发烧友。

体验下来，山灵的确是不出错的选择。弗西 Merak 设计绝赞，它甚至还能翻录，美中不足的是没有电池；飞傲子品牌雪漫天的 BeatBox 体型略大，比较适合纯入门，它跟少数派还有合作，顶盖面板可以替换，但更像玩具；其他的我都有尝试，但价格太贵，就不考虑了。

返程后我心里踏实下来，没选错 CD 机。

周日山灵 EC Play 到货，质感颇佳，而且尺寸刚好和一张 CD 差不多。等到我自己购买的光碟到货后，亲手取出，这是我首次感到专辑是可触摸的。我从头到尾听完一整张，心想和听流媒体真有些区别，更沉浸，更连贯，不容易被打断。

电子设备当然也能完整地听完一整张专辑，但大多时候是当作背景音乐，我会专心做自己的事情。而听专辑，它不是搜索点击播放，我需要找到一张专辑，打开 CD 盒，再放入托盘，按下播放键，这会给你一种仪式感，告诉身体和大脑：我要好好听音乐了。

再从拥有的角度来说，CD 的好处是只要它不损坏，那我就永远拥有这张专辑；而流媒体会受到版权和会员期限的限制，你购买的只是播放权，除非你宁愿大费周章下载文件到本地。

目前我购买了六张 CD，发现比音乐软件的会员费还要贵。这些 CD 里，有三张张国荣，一张孙燕姿，以及今天刚到货的两张罗大佑。其间有正版也有盗版，但光凭音质，我的确听不出任何区别，不过有两张盗版会卡碟，商家给我的建议是擦一下，结果擦完后还是老样子，那就作罢。

![1779200589170.jpg](https://img.liangmouyin.com/2026/05/c0066bab58f21831805b7b901b253c2c.jpg)

购买 CD 不像在流媒体那样把专辑加入收藏那般简单，需要反复对比。我最近想买 Zazazsu 的《第一千夜》，我要看哪个平台有售，哪个价格最低，顺便看看闲鱼有没有人卖二手，如果有，少不得讨价还价。

经济上的限制，也让我不会购买太多 CD。由于选择太少，也就意味着我只能购买我能够听完一整张 CD，而不是刷短视频看到哪张专辑有意思，就火速下单，最后能够变成实体的，也就那些值得反复听的专辑。

弊端可能是丧失听新歌、扩充曲库的机会。这话不假，但听音乐本来也可以在一小间房子，听那有限的几首；在流媒体漫游，我会频繁切歌，即便听到好的音乐，也只是当作背景音乐，心思全不在音乐上。

即便真要拓展曲库，也应该是横向尝试更多风格，时间线上再往过去扒拉一些专辑，而不需要听今年突然流行的音乐——因为它们的生命力可能也就只有一年。

我想过，哪怕一两年后，我对 CD 不感兴趣，那折价出二手或者送人，我都能接受。因为我现在触摸着 CD 机，以及听 CD 是喜悦的，拒绝 CD 机的理性理由有无数个，但也难敌「我喜欢」这三个字。

2026 年，又入一个新坑，接下来就能感受挑 CD、买 CD 和听 CD 的乐趣了。
