经朋友提醒，我才发现已经有四五个月没有更新我的博客「Now」页面了。

因此我昨天上去更新了一下，但发现还有很多事没说出来。以及我发现，我的身上有一些转变，这种转变也不是说好或者坏，只是单纯发生了。

## 在遥远的互联网上

我在上周末将博客重新设计了一下。几乎我只是动了一张嘴，再经由 Claude Code 来进行修改。最后的成果我也很满意，但是后面还有不少 bug，以及我的需求变来变去，中间也经过了很多次的反复切磋和修改。如今的模样，我还是挺满意的。

顺带能聊聊我使用 AI 的变化。

现在我已经开始从 Gemini 切换到 Claude，甚至还为此充了钱。原因是我发现 Gemini 太过讨好我，聊到最后，我发现自己还是在回声里面打转。

中间我也切换了一下 GPT，但后来还是觉得 Claude 比较符合我的口味，因为它不哄着我，说话很冷峻、很直接，像一个诤友。后来用上 Claude Code 之后，因为订阅了 Pro 会员可以直接用，在客户端或者是终端都能用。

我更觉得这个会员开得值得。在客户端用着很舒服，它比终端的好处是界面看着更优雅。除了作为我的博客助手，我还让它打开工作文件，给我一些建议。

昨天我看朋友整了周刊，鄙人也有几分想法。只是周刊太快，月刊太慢，我干脆折中，做个半月刊。我让 Claude Code 帮我建文件夹时（是的，从新建文件夹开始），它命名为 `biweekly`，我只能看懂一个 week，跟星期有关系，以为 Claude 帮生造了一个英文单词作为名字，查后方知它的名词含义：双周刊，每两星期出刊的刊物。

又学到一个单词。

## 回到这具肉身

从线上到线下，也有几点能聊聊。最直接的是，我出门上班开始不戴口罩了。

自从来到深圳开始，我几乎每天上班通勤都会戴着口罩。因为地铁人口密集，全部困在小小的车厢里，戴着有备无患，反正我戴着不难受。尤其是在流感高发期的时候，那口罩肯定是要戴的。

现在我也说不清楚是什么原因，我就不戴了。可能是松懈，或者是觉得已经戴了两三年，试试另一种生活。至于我「阳过」没有，我没有买试纸来测过（毕竟不便宜），因此也不知道。康哥曾对我戴口罩表示困惑，毕竟疫情已经过去三年，为什么一个人还傻傻地每天戴？让他猜去吧。

尽管呼吸自由，可我的腿部又开始不争气了。

从上周开始，我就感觉左脚生疼。先是脚底，然后脚背、脚踝，连着小腿也疼。我的推测是之前的鞋带系得太紧，导致我的脚被挤变形了；走路姿势一扭曲，带着肌肉和筋膜都在用力，最后就疼了。这一结果是，走路极慢，快起来就像瘸子，因为疼。

现在已经疼了三四天，看看接下来几天会不会好一点。

## 顺带改造环境

我还把床给立起来了。

之前睡了几个月地板，后来因为深圳入冬，气温骤降，晚上冻得不行，我就只能从地板回到床垫。如今深圳的气候已经开始暖和，那么睡回地板也是顺理成章的。

第一天睡完之后发现腰酸背痛，第二天就没什么感觉了。主要是这么做会让空间多出来很多，走路会更方便一点，还能打赤脚。因为如果占了很大一部分床位的话，我觉得打赤脚是没什么必要的；如今就能横着走了。

说回脚疼的事。早上因为出门晚了10分钟，导致我去赶地铁上班的时候慢了不少，最后卡在最后一分钟打上卡。可这么一来我就没早餐吃了。

我之前吃早餐一直都是吃麦当劳，因为买了个早餐卡：便宜的时候，花五六块钱就能吃个汉堡加豆浆；想吃点好的，就会点个十几块钱的，多两根香肠。

只是我连续吃了两三个月，同事都震惊你怎么还在吃麦当劳，我也想换换胃口，但想要更方便的早餐。最后我的答案是：买一些能放在办公室吃的东西。

目前我买了即食燕麦片、纯牛奶和纯坚果干（小袋包装的）。这三个加起来，就算一顿早餐，要不了几分钟就够。我算了一下成本，并不算高，一顿不超过6块钱，比之前吃麦当劳还便宜。

可能是因为我对食物的耐受力比较强（味觉和嗅觉薄弱），我吃东西不挑，只要能入口就行。等东西到齐了，看看能不能作为长期早餐。

这些是变化还是迭代？这并不重要，想来只是生活节奏自然而然的变迁，也许过段时间就回去了，又或者朝着另一个方向走，这谁知道呢？
