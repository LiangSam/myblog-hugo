终于把塔罗牌网站[牌间](https://tarot.liangmouyin.com)推出来了。

起因有点偶然——昨天朋友问我 Claude Design 的问题，我顺手打开试了试，想着既然如此，干脆把一直想做的塔罗牌网站给做了。之前一直卡在风格选不定，做着做着冒出一个念头：要不试试《纪念碑谷》那种风格？于是就这么定下来了。

图片部分用 ChatGPT 生成 22 张大阿卡纳，过程比想象中麻烦——头几张风格挺对，后来开始漂移，反复试错，为此还专门买了 Plus 会员，才算把全套凑齐。踩过一个坑值得记一下：不要让 AI 一开始就把边框和名字一起画进去，一旦画错了后面很难改回来；正确的做法是先生成纯图片，再单独写一个装裱器去适配。

开发上，Claude Code 帮我搭框架、写功能，我让它全程写日志记录进度；Claude 限额之后切到 ChatGPT Codex，它能根据日志直接接着干，然后不断优化功能，包括但不限于字体、颜色、牌阵图片、抽牌方式、图片大小、增加 About 页面、怎么连接 DeepSeek 的API、。最后通过 GitHub 部署到 Cloudflare，API key 也做了保护，没有暴露在前端。

从想法到上线，一天搞定，这就是 vibe coding 的乐趣啊。虽然没啥用，但挺开心的。
